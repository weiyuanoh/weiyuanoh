1. Start with a market idea or structural intuition  
2. Translate it into a testable framework  
3. Build the data, model, or backtest pipeline

5. Refine or discard and repeat

## Selected Research

### Bayesian Inverse Problems with Adaptive Finite Element Methods

My undergraduate thesis in applied mathematics, focused on recovering unknown parameters in a PDE-governed system using **Bayesian inference** and **adaptive finite element methods**.

This work brought together mathematical modelling, numerical methods, and scientific computing: starting from the governing PDE, building the numerical approximation, and studying inference under computational and discretization constraints.

**Research areas**
- PDE-based modelling  
- Bayesian parameter inference  
- Adaptive finite element methods  
- Mesh refinement and error estimation  
- Numerical approximation and computational accuracy  
- Scientific computing in Python  

This remains one of my flagship pieces of work because it reflects the kind of problems I enjoy most: mathematically structured, computationally demanding, and built carefully from first principles.

**Tools**  
`Python` `NumPy` `SciPy`

### Kalman Filter Research for Statistical Arbitrage

A statistical arbitrage research effort on **dynamic hedge ratio estimation** and **mean-reversion signal design**, comparing a **standard random walk state-space Kalman filter** with an **ECM-driven Kalman filter** motivated by cointegration and error-correction dynamics.

This research was as much about **model structure** as performance: understanding when a purely statistical state-space formulation is sufficient, and when a more economically informed specification gives a better representation of the relationship between assets.

**Research areas**
- Random walk state-space modelling  
- ECM-driven Kalman filtering  
- Dynamic hedge ratio estimation  
- Cointegration and error-correction logic  
- Signal construction from residual and innovation dynamics  
- In-sample and out-of-sample testing  
- Walk-forward validation  
- Leakage control and signal alignment  
- Trade diagnostics, attribution, and robustness checks  

**Tools**  
`Python` `pandas` `NumPy` `statsmodels` `matplotlib`
